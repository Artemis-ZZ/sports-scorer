# 垒球+户外运动 体育成绩管理系统

中国传媒大学体育部 — 张嘉雯

## 功能

- 📋 **学生管理**: 导入 Excel 学生名单，手动添加/删除
- 📅 **考勤**: 按课程时间点名，支持旷课/病假/事假/公假/见习
- 💪 **身体素质**: 50m跑/800m(女)/1000m(男)/立定跳远/仰卧起坐/引体向上，自动对照评分表算分
- 🎯 **专项**: 垒球(专项一20分+专项二20分) / 户外运动(专项一30分+专项二10分)
- 🏆 **专项赛事**: 竞赛一(5分) + 竞赛二(5分) + 校级赛事(10分)，上限10分
- ☀️ **阳光长跑**: 默认10分，下拉选择
- 🔍 **查询**: 查看单个学生全部成绩
- 📊 **总览**: 成绩总览 + 统计分析 + Excel 导出

## 评分构成 (总分100)

| 项目 | 分值 |
|------|:----:|
| 考勤 | 10分 |
| 身体素质 (4项) | 30分 |
| 专项一 | 20分(垒球) / 30分(户外) |
| 专项二 | 20分(垒球) / 10分(户外) |
| 专项赛事 | 10分 (上限) |
| 阳光长跑 | 10分 |

## 部署

1. 在 GitHub 创建仓库 `sports-scorer`
2. 推送代码并启用 GitHub Pages (Settings → Pages → Source: main branch)
3. 访问 `https://artemis-zz.github.io/sports-scorer/`

## 本地运行

```bash
cd sports-scorer
python3 -m http.server 8082
# 打开 http://localhost:8082
```
